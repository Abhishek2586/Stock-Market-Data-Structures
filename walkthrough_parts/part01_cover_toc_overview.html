Section 01

Project Overview

StockADS — Stock Market Data Aggregator is a real-time stock market data management system that demonstrates 5 core Advanced Data Structures through a practical financial application. Each module solves a specific stock market problem using a different advanced data structure — Hash Table, Priority Queue, AVL Tree, Heap, and Trie.

  • A C backend implementing 5 data structure modules (member1_hashtable.c through member5_trie.c)
  • A Python web server (server.py) that compiles C code and runs it live via HTTP API
  • A premium HTML/CSS/JS frontend with 6 interactive pages — Overview, Hash Table, Priority Queue, AVL Tree, Heap, Trie
  • A shared data header (stock_data.h) with 12 sample stock records used across all modules

Why This Covers Advanced Data Structures

This project implements 5 advanced data structures from the ADS syllabus — AVL Trees (Unit 1), Priority Queues and Heaps (Unit 2), and Tries (Unit 3). Each data structure solves a real-world stock market problem, demonstrating practical applications of theoretical concepts. The Hash Table provides foundational O(1) data access, while the remaining four structures directly map to Units 1–3 of the ADS curriculum.
