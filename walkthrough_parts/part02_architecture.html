Three-Layer Architecture

The system follows a three-layer architecture: a premium web frontend communicates with a Python server, which compiles and executes the C data structure modules on demand.

How the Frontend Renders Data Structures

Each page in the frontend has two modes of operation:

  1. Interactive JavaScript Mode — The frontend implements each data structure in JavaScript for real-time UI interaction (insert, search, visualize)
  2. Live C Execution Mode — Click "Run Live" to compile and execute the actual C code on the server, showing raw terminal output

This dual approach lets users interact visually with the data structures while also seeing the actual C implementation output.

Architecture Highlights

Real-time C compilation: The Python server compiles all 5 C files + main.c into a single binary using GCC. Each module is executed by sending the menu choice via stdin. CORS-enabled API: The server exposes /run?module=N and /compile endpoints. No external dependencies: Pure C with standard libraries only (stdio, stdlib, string, time, math).
