The shared header defines the common StockRecord structure and 12 sample stock entries used across all 5 modules. This ensures data consistency and demonstrates modular C programming.

File 2: main.c — Unified Entry Point (64 lines)

Single entry point with a menu-driven CLI. Dispatches to each module via switch-case. Supports running individual modules or all 5 sequentially with option 6.
