Topic 4: Heap — Max/Min Ranking

A Heap is a complete binary tree where each parent node satisfies the heap property. A Max-Heap has every parent ≥ its children (largest at root). A Min-Heap has every parent ≤ its children (smallest at root). Our project uses both simultaneously — a max-heap for top gainers and a min-heap for top losers.

Dual Heap Architecture

Stock arrives with change_percent: ┌─────────────────────────────────────────────┐ │ if change_percent >= 0 → MAX-HEAP (Gainers)│ Root = highest gainer │ if change_percent < 0 → MIN-HEAP (Losers) │ Root = biggest loser └─────────────────────────────────────────────┘ Extract-Max from Gainer Heap → "Top Gainer" (e.g., TSLA +12.3%) Extract-Min from Loser Heap → "Top Loser" (e.g., GOOGL -5.2%)

Heap Insert — Heapify Up

// FROM member4_heap.c
void heap_insert(Heap* h, const char* sym, double price, double chg, long vol) {
    if(h->size >= MAX_HEAP_SIZE) return;

    // Place at end of array
    int i = h->size++;
    strcpy(h->data[i].symbol, sym);
    h->data[i].price = price;
    h->data[i].change_percent = chg;

    // Bubble up: compare with parent, swap if heap property violated
    while(i > 0) {
        int parent = (i - 1) / 2;
        if(h->is_max_heap) {
            // MAX-HEAP: parent should be >= child
            if(h->data[parent].change_percent < h->data[i].change_percent) {
                heap_swap(&h->data[parent], &h->data[i]);
                i = parent;
            } else break;
        } else {
            // MIN-HEAP: parent should be <= child
            if(h->data[parent].change_percent > h->data[i].change_percent) {
                heap_swap(&h->data[parent], &h->data[i]);
                i = parent;
            } else break;
        }
    }
}

Heap Extract — Heapify Down

// FROM member4_heap.c — Extract root (max or min) and restore heap
HeapStock heap_extract(Heap* h) {
    HeapStock top = h->data[0];               // Save root (best element)
    h->data[0] = h->data[--h->size];           // Move last to root

    // Push down: compare with children, swap with appropriate child
    int i = 0;
    while(1) {
        int best = i;
        int left  = 2*i + 1;
        int right = 2*i + 2;

        if(h->is_max_heap) {
            if(left  < h->size && h->data[left].change_percent  > h->data[best].change_percent)  best = left;
            if(right < h->size && h->data[right].change_percent > h->data[best].change_percent) best = right;
        } else {
            if(left  < h->size && h->data[left].change_percent  < h->data[best].change_percent)  best = left;
            if(right < h->size && h->data[right].change_percent < h->data[best].change_percent) best = right;
        }

        if(best == i) break;
        heap_swap(&h->data[i], &h->data[best]);
        i = best;
    }
    return top;
}
