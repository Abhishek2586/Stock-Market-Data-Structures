Anonymous struct array! Instead of defining a named struct type, we define the structure inline and immediately create an array of it. This is a C shorthand.

Each element has two fields: symbol (string) and price (double). 18 stocks total.

Faculty Answer: "This is an anonymous struct declaration with an initializer list. We define the structure and instantiate an array simultaneously. Each element is a compound literal with a symbol string and price value. 18 stock entries are initialized."
Line 95
int n = sizeof(stocks)/sizeof(stocks[0]);
Classic C trick to calculate array length!

sizeof(stocks) = total bytes of the entire array
sizeof(stocks[0]) = bytes of ONE element
Dividing gives the NUMBER of elements: 18

Why not just write 18? Because if we add or remove stocks, this automatically adjusts. No manual counting needed!

Faculty Answer: "This computes the array length at compile-time using the sizeof operator. It divides the total array size by the size of a single element, making the code maintainable — adding or removing entries doesn't require updating a hardcoded count."

Tests exact search with 4 symbols. "XYZ" is intentionally NOT in the trie to show a failed search.

%-6s — left-aligned string padded to 6 characters (for neat formatting).

Faculty Answer: "The ternary operator provides inline conditional string selection. We test with both existing symbols (AAPL, GOOGL, MSFT) and a non-existent one (XYZ) to demonstrate both successful and failed search cases."
