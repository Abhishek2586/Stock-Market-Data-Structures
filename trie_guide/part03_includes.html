Line 10: Including the shared header

#include "stock_data.h"
Line 10
#include "stock_data.h"
#include is a preprocessor directive — compiler ko bolta hai ki "is file ka saara content yahan paste kar do compile se pehle".

Double quotes "stock_data.h" means look in the CURRENT directory first (not system folders). This is our project's shared header file.

What does stock_data.h contain?
• #include <stdio.h> — for printf, scanf
• #include <stdlib.h> — for malloc, calloc, free
• #include <string.h> — for strcmp, strcpy, strncpy
• #include <time.h> — for time functions
• #define MAX_SYMBOL_LEN 10 — max length of stock symbol string
• Common struct definitions used across all 5 member files

Faculty Answer: "This includes our shared project header which provides common definitions like MAX_SYMBOL_LEN, standard library includes (stdio, stdlib, string), and shared data structures used across all five modules."

Line 11: Including ctype.h

#include <ctype.h>
Line 11
#include <ctype.h>
Angle brackets <ctype.h> means look in SYSTEM include directories (not our project folder).

ctype.h stands for "Character TYPE Header". It provides functions to check and convert characters:
• toupper(c) — converts lowercase 'a' to uppercase 'A' (we use THIS one!)
• tolower(c) — converts uppercase to lowercase
• isalpha(c) — checks if character is a letter
• isdigit(c) — checks if character is a digit

Why do we need it? Because our char_idx() function uses toupper() to convert any character to uppercase before mapping it to an array index (0-25). This makes our trie case-insensitive.

Faculty Answer: "We include ctype.h for the toupper() function, which ensures case-insensitive character handling. Whether the user types 'aapl' or 'AAPL', it gets converted to uppercase before trie operations."
⚠️ Difference: "quotes" vs <angle brackets>

#include "file.h" — Searches in YOUR project directory first, then system directories. Used for YOUR OWN header files.
#include <file.h> — Searches ONLY in system/standard library directories. Used for STANDARD C library headers like stdio.h, stdlib.h, ctype.h.
