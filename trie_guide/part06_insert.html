Lines 33–46: trie_insert() — Full Breakdown

This function inserts a stock symbol into the trie, character by character. This is THE core operation.

static void trie_insert(TrieNode* root, const char* symbol, double price) {
    TrieNode* cur = root;
    printf("  [INSERT] %s ($%.2f)\n", symbol, price);
    for(int i = 0; symbol[i]; i++) {
        int idx = char_idx(symbol[i]);
        if(idx < 0 || idx >= ALPHABET_SIZE) continue;
        if(!cur->children[idx]) cur->children[idx] = trie_new_node();
        cur = cur->children[idx];
    }
    cur->is_end = 1;
    strncpy(cur->symbol, symbol, MAX_SYMBOL_LEN - 1);
    cur->last_price = price;
    trie_words++;
}

Line 34
TrieNode* cur = root;
Creates a cursor pointer starting at root. We'll walk this pointer down the trie. We don't want to modify root itself (we need root to stay at the top), so we use a separate pointer cur to traverse.

Think of it as: root = ghar ka address (permanent), cur = aapka GPS location (moves as you walk).
Line 35
printf(" [INSERT] %s ($%.2f)\n", symbol, price);
Prints a log message showing what's being inserted. %s = string (symbol), $%.2f = dollar sign + float with 2 decimal places (price). \n = newline. This is just for visual output, not part of the algorithm.
Line 36
for(int i = 0; symbol[i]; i++) {
THE MAIN LOOP — iterates through each character of the symbol.

• int i = 0 — start from first character
• symbol[i] — this is the CONDITION. In C, strings end with a null terminator '\0' (ASCII 0). When we reach '\0', the condition becomes 0 (false) and the loop stops!
• i++ — move to next character

For "AAPL": i=0→'A', i=1→'A', i=2→'P', i=3→'L', i=4→'\0' (STOP)

Faculty Answer: "This iterates over each character of the symbol string. The loop condition symbol[i] evaluates to false (0) when it encounters the null terminator '\\0', which marks the end of the C string. This is a standard C idiom for string traversal."
Line 37
int idx = char_idx(symbol[i]);
Converts the current character to an array index (0-25). For 'A'→0, for 'P'→15, for 'L'→11. This index tells us WHICH child pointer to follow.
Line 38
if(idx < 0 || idx >= ALPHABET_SIZE) continue;
Validation/Safety check! If the character is not a letter (like a digit or symbol), char_idx would return a value outside 0-25 range. This check SKIPS such invalid characters using continue (jumps to next loop iteration).

Faculty Answer: "This is a bounds-checking guard clause. If the character maps to an index outside the valid range [0, 25], we skip it using continue. This prevents array out-of-bounds access and handles any non-alphabetic characters gracefully."
Line 39
if(!cur->children[idx]) cur->children[idx] = trie_new_node();
THE CREATION STEP!

!cur->children[idx] = if child at this index is NULL (doesn't exist yet)...
cur->children[idx] = trie_new_node() = ...create a new node and link it!

If the path already exists (from a previously inserted word sharing this prefix), this line does NOTHING — we reuse the existing node. This is how tries share prefixes!

Example: After inserting "AMD", the path A→M exists. When inserting "AMZN", A and M nodes already exist, so no new nodes needed for those — only Z and N are new.

Faculty Answer: "This implements lazy node creation — we only allocate a new node if the path doesn't already exist. If a previous insertion created this path (shared prefix), we reuse the existing node. This is the key mechanism for prefix sharing in tries."
Line 40
cur = cur->children[idx];
Move the cursor DOWN to the child node. We've processed one character, now move to the next level of the trie. Next iteration of the loop will process the next character starting from this new position.
Lines 42–45
cur->is_end = 1;
strncpy(cur->symbol, symbol, MAX_SYMBOL_LEN - 1);

After the loop, cur points to the LAST node of the word. Now we mark it:

• is_end = 1 — "a complete word ends here"
• strncpy — copies the symbol string safely (max MAX_SYMBOL_LEN-1 chars to prevent buffer overflow). Unlike strcpy, strncpy takes a max length parameter for safety.
• last_price = price — store the stock price
• trie_words++ — increment word counter

Faculty Answer: "After traversing/creating the path for all characters, we mark the terminal node as a word boundary by setting is_end=1, store the complete symbol string using strncpy for buffer-safety, store the associated price, and increment our word counter."
